FLEX ITEMS

FLEX ITEM

Como demonstrado na página anterior, o CSS FlexBox consiste em um Flex Container, que é o elemento Pai, e em Flex Items, que são os filhos deste primeiro.
A partir das funcionalidades do Flex Container, e de acordo com seus eixos principal e transversal, os Flex Items podem ser posicionados de forma agrupada.
Mas, os próprios Flex Items podem ser manipulados diretamente, e não apenas em termos de posicionamento, mas também de tamanho e capacidade de expansão e encolhimento.


Propriedades DEFAULT dos Flex Items

Por padrão, o tamanho inicial dos elementos é definido a partir do seu conteúdo interno. Portanto, o "flex-basis" é "auto".
Por padrão, não há expansão no eixo principal, sendo obedecido o width do elemento ou o flex-basis. Portanto, o "flex-grow" é "0".
Por padrão, os itens podem ser reduzidos, a partir de seu tamanho inicial, para caber no Container. Portanto, o "flex-shrink" é 1.
Por padrão, os itens serão esticados para preencher o espaço disponível no eixo transversal. Portanto, o "align-self" é "stretch".

FLEX BASIS

A propriedade flex-basis define o tamanho inicial dos elementos, em unidades de pixel, antes que o espaço remanescente seja redistribuído.
Seu valor padrão "auto" define que se os elementos não possuírem um tamanho já definido, com width, as dimensões dos seus conteúdos serão utilizadas como medida.
Também pode ser definido com valores em pixel, %, e outros. Mas, neste caso é importante enfatizar que as medidas usam como referencial o eixo principal do Flex Container.
Quando o Flex Grow é declarado com um valor igual ou maior que 1, o Flex Basis, se omitido, por padrão será 0.
Se o Flex Basis for declarado com o valor 0, sem outra declaração de dimensionalidade, o efeito será igual ao de atribuir nulo ao width.
Os valores possíveis são: auto, medida px, medida % e 0.

FLEX GROW

Esta propriedade define a capacidade de um Flex Item crescer ao longo de um espaço disponível no eixo principal do Flex Container.
Por padrão seu valor é 0, o que significa que os elementos obedecem ou a um width declarado, ou ao valor do Flex Basis.
Se o Flex Grow for definido como 1, o elemento irá ocupar 100% do espaço disponível no eixo principal.
Se existir mais de um Flex Item, definido com Flex Grow igual a 1, ambos tentarão igualmente ocupar o espaço disponível, mas respeitando o tamanho de seus conteúdos internos.
Se um Flex Item for definido com Flex Grow igual a 1, e outro com Flex Grow igual a 2, este último irá ocupar o dobro do espaço do primeiro. Sendo assim, as medições podem ser relativas.
Atenção: Se o Flex Grow for definido como 1, a propriedade Justify-content não poderá ser utilizada, pois desloca, em conjunto, todos os itens ao longo do eixo principal apenas se houver espaço disponível.
Os valores possíveis são: 0, 1 e números maiores que 1.

FLEX SHRINK

Esta propriedade define a capacidade de um Flex Item reduzir seu tamanho, a partir de um inicial, definido com uma declaração Flex Basis ou width, para caber no Flex Container.
Por padrão, seu valor é 1, o que significa que todos os itens podem reduzir seus tamanhos para caber no Container, se necessário.
Se um elemento for definido com Flex Shrink de valor 0, seu tamanho não irá reduzir. Assim, se o Container Pai, ou Flex Container se tornar menor que o próprio Flex Item -seu filho-, este último irá ultrapassar os seus limites.
Se, por exemplo, um Flex Item tiver Flex Shrink declarado como 0, e outro como 1, com a redução do Flex Container o segundo irá reduzir, e o primeiro não irá.
Além disso, como o Flex Grow, o Flex Shrink admite medições relativas.
Os valores possíveis são: 0 e 1.

ORDER

Esta propriedade permite ordenar os Flex Items.
Funciona em relação ao eixo principal, e do menor para o maior valor. Assim, por exemplo, um Flex Item com Order 1 estará na frente de outro com Order 2.
Os valores possíveis são: todos.

ALIGN SELF

Esta propriedade permite sobrescrever no item o comportamento que foi definido pela propriedade align-items. Ou seja, o deslocamento será de um só Flex Item, e no eixo transversal.
Os valores possíveis são: auto, stretch, flex-start, flex-end, center e baseline.
