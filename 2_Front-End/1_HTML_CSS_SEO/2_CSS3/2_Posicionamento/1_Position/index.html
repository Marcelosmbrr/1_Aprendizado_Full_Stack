Position: Static

Esse é o posicionamento DEFAULT de todo elemento html.

As propriedades de posição top, bottom, left e right não podem ser utilizadas.

Position: Relative

O elemento respeita o fluxo do documento, mas pode ser deslocado com as propriedades de posição.

Como podemos notar, quando o elemento for deslocado, o seu lugar original no fluxo é preservado.

Ancestral mais próximo do elemento absolute

Esta caixa está posicionada com relative, e servindo de referencial para o elemento absolute.

Position: Absolute

O elemento ignora o fluxo do documento, e se posiciona em relação ao seu ancestral mais próximo que estiver posicionado. Admite as propriedades de posicionamento.

Se não existir um ancestral posicionado, o elemento irá considerar como referêncial unicamente o body.

Além disto, o elemento absolute, diferentemente do elemento relative, não deixa espaço no fluxo do documento.

Position: Fixed

Um elemento posicionado de forma fixa fica posicionado em relação ao viewport, ou seja, em relação à tela. Admite as propriedades de posicionamento.

Também não deixa nenhum espaço para o elemento no fluxo do documento.

Se existir rolagem da página (scroll), o elemento vai sempre ficar no mesmo lugar em relação à tela, não acompanhando a deslocação da página.

Position: Sticky

O posicionamento aderente é um híbrido de posicionamento relativo e fixo.

O elemento é tratado como posicionado relativo até que cruze um limite especificado, ponto em que é tratado como posicionado fixo.

Os valores de posição são usados para definir o breakpoint para a posição fixa.

Neste caso, usamos top: 0, para que o elemento se torna fixo no topo quando rolarmos a página.
