Guia: Origamid Grid Layout

Um container marcado com Grid é chamado de "Grid Container", e aquilo que existe dentro dele de "Grid Item".

Utilizar o posicionamento Grid implica em posicionar bidirecionalmente os "Grid Items" de um "Grid Container". Um grid é uma grade de linhas e colunas.

Importante: Os Grid Items e o grid/grade são coisas distintas! Os Grid Items se formam e posicionam de acordo com a grade, e que existe de forma implícita. Por isso, é possível definir uma grade sem Grid Items.
