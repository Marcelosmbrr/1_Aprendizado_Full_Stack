Guia: Origamid Flexbox

Um container marcado com Flexbox é chamado de "Flex Container", e aquilo que existe dentro dele de "Flex Item".

Utilizar o posicionamento Flex implica em posicionar unidirecionalmente os "Flex Items" de um "Flex Container".

O "main axis" é o eixo principal escolhido. Será o eixo X se o Flex Direction for "row", e será o eixo Y se for "column".

O "cross axis" é o eixo transversal, isto é, o alternativo ao principal escolhido.
