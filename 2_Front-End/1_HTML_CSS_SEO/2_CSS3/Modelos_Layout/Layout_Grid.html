  • Esta configuração é: grid-template-rows = 120px 1fr 150px e grid-template-columns = 250px 1fr.

  • O template rows define a quantidade de linhas, e o template columns define a quantidade de colunas por linha. Então são 3 linhas com duas colunas cada.

  • Por padrão os GRID items são alocados de acordo com o fluxo do HTML e as células do GRID - um item para cada célula. Eles seriam o header, main, aside e footer.

  • O GRID possui linhas virtuais que delimitam as células, e elas são utilizadas como os referenciais para a customização espacial dos GRID items.

  • As linhas são o fator crucial. Como dito, por padrão, um GRID item ocupa uma célula, isto é, o espaço entre duas linhas, mas pode ser configurado para ocupar mais de uma.

  • A contagem das linhas virtuais começa nas extremidades. Cada linha aqui, possuindo duas colunas, possui três linhas virtuais, contando da esquerda para direita, por exemplo.

  • O header começa na linha 1 do eixo horizontal, e termina na 3, ocupando todo o espaço nesse sentido, assim como o footer; o aside começa na linha 2 do eixo Y, e termina na 3; o main se auto adaptou ao espaço restante.
