HG WEATHER API

Esta aplicação utiliza uma API de metereologia da HG Brasil. É possível recuperar dados metereológicos, do tempo presente, de municípios brasileiros.

Foi utilizada uma key de uso limitado, o que pode vir a demandar a geração de uma nova para a utilização da API.
