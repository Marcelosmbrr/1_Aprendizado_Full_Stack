Box Sizing

As duas caixas tem 500px de largura, e 500px de altura, mas, ainda assim, a primeira é muito maior do que a segunda.

Isso ocorre porque o padding e a borda do elemento estão sendo adicionados às dimensões base dos elementos.

Com "box-sizing: box-content", que é o valor default, isso ocorre. Assim, a primeira caixa possui 500px de largura + 200px de padding + 5px de borda.

Com o valor "box-sizing: border-box", as propriedades que adicionam pixels, como padding e border, se acumulam sem ultrapassar o valor do width.
