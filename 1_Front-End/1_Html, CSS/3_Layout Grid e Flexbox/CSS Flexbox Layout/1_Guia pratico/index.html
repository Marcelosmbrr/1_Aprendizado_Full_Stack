FLEX CONTAINER

DISPLAY FLEX

A área de um documento que faz uso do flexbox é chamada de Flex Container. Para criar essa estrutura, define-se o valor da propriedade display do elemento representado pelo contêiner como flex ou inline-flex. Desse modo, os elementos-filhos desse contêiner tornar-se-ão do tipo flex.
Assim, ao utilizar a propriedade Flexbox, deve-se manter em mente o fato de existir tanto o próprio Flex Container, quanto seus filhos, que serão Flex Items.
As propriedades manipuláveis, disponibilizadas pelo display FlexBox, existem tanto para o Flex Container quanto para os Flex Items.


Propriedades DEFAULT do Display Flex

Por padrão, o eixo principal é o horizontal. Portanto, o "flex-direction" é "row".
Por padrão, o alinhamento se dá na borda inicial (superior esquerda) do eixo principal. Portanto, o "justify-content" é "flex-start".
Por padrão, os itens serão esticados para preencher o espaço disponível no eixo transversal. Portanto, o "align-items" é "stretch".
Por padrão, não há expansão no eixo principal. Os elementos ocupam o espaço de seu conteúdo. Portanto, o "flex-grow" é 0, e o "flex-basis" é "auto".
Por padrão, os itens podem ser reduzidos para caber no Container. Portanto, o "flex-shrink" é 1.
Por padrão, os itens do container tentarão se ajustar em uma única linha dentro do container pai. Portanto, o "flex-wrap" é "nowrap".

FLEX DIRECTION

A propriedade flex-direction deve ser aplicada ao container e define o eixo/fluxo de exibição em que os elementos serão organizados.
Os valores possíveis são: "row", "row-reverse", "column" e "column-reverse".
Se o fluxo for definido como "row", o eixo principal será o horizontal, e se for definido como "column", será o vertical.
A diferença de "row" e de "row-reverse", e de "column" e "column-reverse" é a ordem de distribuição dos elementos em si, que são inversas.
Assim, por exemplo, se com "row" a posição inicial dos elementos é a borda superior esquerda, do Flex Container, com "row-reverse" será na superior direta.

JUSTIFY CONTENT

A propriedade justify-content define o alinhamento dos itens ao longo do eixo principal do container.
Igualmente, só funciona se houver espaço disponível no Flex Container.
Os valores possíveis são: "flex-start", "flex-end", "center", "space-between" e "space-around".

ALIGN ITEMS

Essa propriedade define como os itens são distribuídos ao longo do eixo transversal do container.
Os valores possíveis são: "flex-start", "flex-end", "center", "baseline".

ALIGN CONTENT

Alinha as linhas do container em relação ao eixo vertical. A propriedade só funciona se existir mais de uma linha de flex-items.
Portanto, a propriedade só funciona se "flex-wrap" estiver definido como "wrap". É na quebra de linha (wrap) que é formada uma nova.
Os valores possíveis são: "flex-start", "flex-end", "center", "space-between" e "space-around".
