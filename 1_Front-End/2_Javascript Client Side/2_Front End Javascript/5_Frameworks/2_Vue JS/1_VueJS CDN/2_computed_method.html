Diferença entre Method e Computed

O Método é inteiramente processado, e um grande número de vezes, sempre que chamado.

O Computado é processado apenas uma vez, e seu retorno é guardado em um cache de acesso rápido. Apenas será reprocessado inteiramente, se suas dependências mudarem, isto é, se os valores envolvidos em sua tarefa forem alterados. E isto, quando ocorre, se dá automaticamente.

1) Data - atributos de soma: a = {{ a }}, b = {{ b }}

2) Retorno do Método -> a + b = {{ somaMetodo() }}

3) Retorno do Computado -> a + b = {{ somaComputado }}

4) Data - atributos contadores: Contador do Método = {{ contadorMetodo }}, Contador do Computado = {{ contadorComputado }}
