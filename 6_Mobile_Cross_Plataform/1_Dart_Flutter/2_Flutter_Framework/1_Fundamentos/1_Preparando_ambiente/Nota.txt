==== O QUE É FLUTTER ====

- Flutter é um kit de desenvolvimento front-end de interface de usuário
- Seu código aberto, e foi criado pelo Google
- Possibilita a criação de aplicativos compilados nativamente
- Atualmente pode compilar para Android, iOS, Windows, Mac, Linux e Web

==== COMO FUNCIONA ====

- Flutter utiliza Widgets 
- Widget é um componente visual para definir a interface de um aplicativo - uma ideia vinda do React
- A ideia central é que você construa sua IU a partir de Widgets/componentes
- Os widgets descrevem a aparência de sua visualização de acordo com sua configuração e estado atuais - existem states como no React
- Quando o estado de um widget muda, o widget reconstrói sua descrição
