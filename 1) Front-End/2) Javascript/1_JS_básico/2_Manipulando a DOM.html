JAVASCRIPT- MANIPULANDO A DOM

DOM (Document Object Model) é uma representação estruturada do documento como uma árvore. O DOM define métodos que permitem acesso à árvore, para que eles possam alterar a estrutura, estilo e conteúdo do documento.

Para entender a síntaxe utilizada em cada exemplo, veja o arquivo fonte.

Exemplo 1 - Eventos

Na tag "button", deste card, incluímos o evento "onclick", que quando disparado realiza alguma tarefa definida na tag de script. Neste caso, criamos a função "mostraAlerta" que dispara o evento "alert()", com o texto "Botão foi clicado", e como parâmetro do evento "onclick()" usamos a própria função. Assim, quando o botão é clicado a função é chamada.
Existem diversos outros eventos pré-definidos.

Exemplo 2 - Método GetElementbyId()

Neste caso, usamos o método para recuperar um elemento da DOM, cujo id seja igual ao parâmetro repassado ao próprio método. Criamos uma variável "seletor" para receber esse elemento, e imprimimos seu valor por meio do "console.log".

Exemplo 3 - GetElementbyClassName() e GetElementbyTagName()

Assim como o anterior, (i) estes métodos recuperam elementos da DOM, (ii) criamos uma variável, e (iii) imprimos no console.log, mas neste caso há um diferencial. Em contraste com o id, tags e classes não definem elementos únicos, sendo possível existir n elementos com determinada tag ou classe.
Sendo assim, estes, quando recuperados, retornam um array.

Exemplo 4 - método querySelector() e querySelectorAll()

Com este método podemos percorrer a DOM para recuperar um elemento, descrevendo a localização como parâmetros do próprio método. Uma variação é o querySelectorAll que retorna todos os elementos que sejam iguais ao parâmetro descrito.

Exemplo 5 - querySelector + Eventos

No primeiro exemplo definimos o evento onClick na própria tag button do HTML, mas, ao invés disto, podemos implementar inteiramente no script - o que é preferível, inclusive.
Basta selecionar um elemento com o querySelector, o atribuindo a uma variável, e, após isto, chamar o evento no próprio script a partir da mesma. Neste exemplo, usaremos novamente o evento "onclick", mas como fora dito, existem outros eventos pré-definidos.
